## Project Overview
This project analyzes Iowa liquor sales data with a focus on the Vodka category.

## Sales Territory
The analysis covers statewide Iowa retail liquor Vodka transactions, with emphasis on
identifying high-volume periods, bulk purchasing behavior, and niche product variants
within the vodka category.
